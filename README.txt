=================ENG=====================

A program for a car service.
It must record information about the owner, the car in the database.
It is planned that a car registration card will be created (the work carried out), 
so that the next time you contact the service, you could add a new record.
Everything should be connected with the employee who made the repairs.

=================RUS=====================

Программа для автосервиса.
Должна записывать в базу данных информацию о владельце, автомобиле.
Планируется, что будет создаваться картачка учёта автомобиля (проведённые работы), 
что при следующем обращении на сервис, можно было бы добавить новую запись. 
Всё должно быть связано с работником, который производил ремонт.
